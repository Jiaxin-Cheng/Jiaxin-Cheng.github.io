PPOCRV5推理模型的使用

PPOCRV5检测和识别模型聚合推理

步骤如下：

  1. 根据requestment.txt文件配置好环境后，下载paddlepaddle-gpu以及pyyaml
  2. 配置tools-> infer文件夹下的的utility.py文件，一般修改image_dir、det_model_dir以及rec_model_dir即可，其中image_dir表示要预测图片的路径，det_model_dir表示检测模型的文件夹，rec_model_dir表示识别模型的文件夹。
  3. 运行predict_system.py文件

PPOCRV5检测模型的推理

  • 命令行推理

python tools/infer_det.py -c configs/det/检测模型对应的yaml文件路径 -o Global.pretrained_model=以pdparams结尾的检测模型路径 Global.infer_img=”推理图片的路径，可以为文件夹”

  • 直接用pycharm运行
    • 在tools文件夹下的program.py文件中修改配置文件的路径为对应模型的配置文件路径
    • 修改配置文件中的pretrained_model、infer_img、save_inference_dir对应的路径
    • 运行tools文件夹下infer_det.py

PPOCRV5识别模型的推理

  • 命令行推理

python tools/infer_rec.py -c configs/rec/识别模型对应的yaml文件路径 -o Global.pretrained_model=以pdparams结尾的识别模型路径 Global.infer_img=”推理图片的路径，可以为文件夹”

  • 直接用pycharm运行
    • 在tools文件夹下的program.py文件中修改配置文件的路径为对应模型的配置文件路径
    • 修改配置文件中的pretrained_model、infer_img、save_inference_dir对应的路径
    • 运行tools文件夹下infer_rec.py
